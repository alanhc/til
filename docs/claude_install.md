# Claude Code 安裝

Claude Code 是 Anthropic 官方的 CLI 工具，可在終端機中進行 agentic 的程式開發。

## 安裝方式
以官方 setup 文件為準：https://code.claude.com/docs/en/setup

常見安裝方式（以官方文件為主）：

```bash
# 透過 npm 全域安裝
npm install -g @anthropic-ai/claude-code

# 安裝後啟動
claude
```

## 備註
- 需要 Node.js 環境（版本需求依官方文件）。
- 首次啟動會引導登入 / 設定 API 認證。
- 若指令或需求有變動，請以官方 setup 文件為準，不要沿用過時指令。
