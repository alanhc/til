# 房（買房筆記）

買房是多數人最大的單筆決策，涉及自住需求與資產配置，需先分清「自住」與「投資」目的。

## 評估重點

- **總價與自備款**：一般自備 2~3 成，其餘貸款；先估自備款是否備妥。
- **貸款**：注意利率、年限、寬限期（只繳息）；每月房貸建議不超過家庭收入的 1/3。
- **地段**：交通、生活機能、學區、未來發展與嫌惡設施（rule of thumb：location 決定保值性）。
- **屋況**：屋齡、格局、坪數（權狀 vs 實坪、公設比）、漏水/管線、社區管理。

## 附加成本

- 一次性：仲介費、代書費、契稅、印花稅、履約保證、裝潢。
- 持有：房屋稅、地價稅、管理費、修繕。
- 出售：土地增值稅、房地合一稅（依持有年限級距）。

## 提醒

- 實際成交行情可查「內政部實價登錄」。
- 看屋多次、不同時段（白天/晚上/雨天）再決定。
