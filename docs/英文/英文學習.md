# 英文學習

累積可理解輸入（comprehensible input）+ 持續輸出練習，比背單字表更有效。

## 方法

- **大量輸入**：聽 podcast、看影片（英文字幕 → 無字幕），閱讀感興趣的文章，重點是「量」與「持續」。
- **情境式記憶**：單字放在句子/情境中記，勝過孤立背誦；善用 Anki 之類的間隔重複（spaced repetition）。
- **主動輸出**：寫作、跟讀（shadowing）、口說；輸出時卡住的地方就是該補的洞。
- **技術英文**：直接讀官方文件、聽技術演講（如 conference talk），同時學語言與領域知識。

## 常用資源

- 影片：YouTube（可調字幕/速度）、TED。
- 字典：優先用英英字典理解語感。
- 工具：Anki（單字卡）、AI 助理（造句、糾錯、對話練習）。
