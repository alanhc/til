- 買進小台指期貨1口

    - 如果漲到20500點
        - 獲利500點 x 50元 = 25000元
    - 如果跌到19500點
        - 虧損500點 x 50元 = -25000元
        - 追加保證金25000元



- long: 購買期貨合約，預期未來價格上漲
    - example: 買進小台指期貨
        原本大盤20000點，買進1口小台指期貨，1點5元
        大盤漲到20500點平倉
        損益：獲利500，500 x 5 = 2500元
        大盤跌到19500點
            『請追加保證金，否則強制平倉』
            
            追加保證金500，500 x 5 = -2500元
            
- short: 賣出期貨合約，預期未來價格下跌
    - example: 賣出小台指期貨
        原本大盤20000點，賣出1口小台指期貨，1點5元
        大盤跌到19500點，獲利500，500 x 5 = 2500元
        大盤漲到20500點，虧損500，500 x 5 = -2500元
