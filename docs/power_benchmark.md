# 功耗 / 效能 Benchmark 工具

測試裝置功耗與效能的 benchmark 工具筆記。

## PCMark

[PCMark](https://benchmarks.ul.com/pcmark10) 以貼近真實使用情境的工作負載（文書、瀏覽、視訊、影像編修）評測整機效能，並提供 **Battery Life** 測試量測續航與耗電。適合評估筆電在日常任務下的效能與功耗表現。

## 其他常見工具

- **3DMark**：偏重 GPU / 遊戲圖形效能。
- **Geekbench**：跨平台的 CPU 單核 / 多核效能。
- 硬體層級量測功耗可搭配電流探針或 power monitor（如 Monsoon）。
