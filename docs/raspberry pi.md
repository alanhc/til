# Raspberry Pi

單板電腦 Raspberry Pi 的筆記。

## 重點

- 用 [Raspberry Pi Imager](https://www.raspberrypi.com/software/) 把 OS（Raspberry Pi OS 等）燒錄到 microSD 卡，開機前可預先設定 SSH、Wi-Fi 與帳號。
- GPIO 腳位可接感測器與周邊，常用於 IoT、home server、自動化與 side project。
- 官方文件：[raspberrypi.com/documentation](https://www.raspberrypi.com/documentation/)。
