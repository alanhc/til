# 畫圖 / 白板工具

常用的線上白板與繪圖工具筆記。

## Miro
- 線上協作白板，功能完整：無限畫布、便利貼、流程圖、看板、範本庫。
- 適合團隊 workshop、腦力激盪、敏捷會議（sprint planning、retro）。
- 商業產品，提供免費方案與付費進階功能。
- 官網：https://miro.com/

## Excalidraw
- 開源、輕量的手繪風格繪圖工具，適合快速畫架構圖、示意圖。
- 手繪（sketchy）風格是其特色，簡潔好上手，可匯出 PNG/SVG。
- 開源可自架，也有官方線上版。
- 官網：https://excalidraw.com/
- 原始碼：https://github.com/excalidraw/excalidraw

## 差異
Miro 偏向完整的團隊協作平台與豐富範本；Excalidraw 偏向輕量、開源、手繪風，適合個人或快速草圖。
