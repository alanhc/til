# Supabase

[Supabase](https://supabase.com/) 是開源的 Firebase 替代方案，以 PostgreSQL 為核心。

## 主要功能

- **Database**：託管的 PostgreSQL，可直接寫 SQL、用 Row Level Security (RLS) 控管權限。
- **Auth**：內建使用者認證（email、OAuth、magic link 等）。
- **Storage**：物件儲存（檔案、圖片）。
- **Realtime**：訂閱資料表變更、即時推播。
- **Edge Functions**：以 Deno 執行的 serverless functions。
- 自動根據資料表產生 RESTful 與 GraphQL API。
