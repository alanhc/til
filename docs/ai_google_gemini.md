# Google Gemini CLI

Google 官方的開源終端機 AI agent，可在命令列直接與 Gemini 模型互動、讀寫檔案與執行工具。

安裝（macOS，透過 Homebrew）：

```
brew install gemini-cli
```

安裝後在終端機執行 `gemini` 即可開始互動；首次使用需以 Google 帳號登入授權。

- 官方文件與原始碼：https://github.com/google-gemini/gemini-cli
