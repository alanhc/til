# 用 Android Flash Tool 幫 Pixel 刷不同版本

Google 官方提供網頁版的 [Android Flash Tool](https://flash.android.com/)，可直接在瀏覽器（Chrome / Edge）幫 Pixel 刷入系統映像檔。

## 步驟重點

- 手機先在「開發者選項」開啟 **USB debugging**，並解鎖 Bootloader（OEM unlocking）。
- 用 USB 連接電腦，於網頁授權裝置後，可選擇要刷入的 **factory image / OTA** 版本。
- 可選版本包含各代 Android 正式版與 Beta / Developer Preview，能降版或刷回特定版本。
- 刷機（尤其解鎖 Bootloader）會清除資料，事前務必備份。
