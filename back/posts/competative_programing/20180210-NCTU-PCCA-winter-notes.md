  輸出：符合條件的第一筆或最後一筆
  作法：每次取中間值，尋找範圍小一半
  big O(log n)

```c
int Binary_Search () {
    int L=0, M, R=INF;
    while (R-L >1) {
        M=(L+R)/2;
        if (check(M)) {
            L=M;
        } else {
            R=M;
        }
    }
    return L;
}
```

## 分治法 Day4

快速冪
合併排序

- 分解 左子 右子
- 解決 數列剩一個元素，甚麼都不用做
- 合併 左右都排序好，甚麼都不用做

最近點對

- 分解 分成 n/2 的左右半邊
- 解決 答案是只剩兩點的距離
- 合併 答案有三種，左半、右半、跨邊點對 把三種可能的答案取 min
